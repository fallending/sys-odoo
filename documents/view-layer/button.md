# 按钮


## 用法

```
<header> 
 <button name="do_toggle_done" type="object" 
   string="Toggle Done" class="oe_highlight" /> 
 <button name="do_clear_done" type="object" 
   string="Clear All Done" /> 
</header>
```


## 解释


```
string:展示在按钮上的文本
type:动作的类型（执行的操作）
name：定义的动作（操作的标识符）
class：一个可选的属性，用于展示Css样式，同HTML规则一样
```
